# CreateFi - Hybrid DEX

CreateFi is a decentralized exchange (DEX) that combines:
- 🔁 AMM (like Uniswap)
- 📘 Order Book (like dYdX)
- 📊 Live charts (like Dexscreener)

## ✅ Features
- Swap tokens with low fees
- Trade using limit/market orders
- View real-time prices and charts
- Earn rewards by providing liquidity

## 🚀 Tech Stack
- Smart contracts (Solidity)
- Frontend (React + Web3)
- Off-chain order matcher (Node.js)
- Charts (Candlestick with PostgreSQL)

## 📂 Folders
- `contracts/` – Smart contracts
- `matcher/` – Order book engine
- `frontend/` – Website
- `backend/` – Price chart builder
